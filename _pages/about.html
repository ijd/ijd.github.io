Colophon

Apps used when writing, building and debugging :

  • Chrome DevTools
  • GitHub Desktop
  • JSFiddle
  • Sublime Text

    • This website is built with the Skeleton Front-End Framework. The code is generated through Jekyll into a static website. Notes are written and published with Markdown. Revisions are saved with Git as the version control system. Hosting is provided by GitHub Pages for free.

    The domain is registered on Gandi.net, which also provides the DNS service to tie everything together. Fonts are loaded from Google Fonts and the font used is called Lato by Łukasz Dziedzic. Statistics are tracked using Google Analytics.
